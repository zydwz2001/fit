Vibe Fitness Pro PRD v3.3 (小米15适配优化版)

1. 核心逻辑
访问鉴权：身体 Tab 设二级加密，必须输入密码 1127 方可进入。
自动换算：身高固定：158cm。BMI 联动：根据实时录入的体重自动换算，公式：$BMI = \text{体重(kg)} / 1.58^2$（禁止手动修改）。
计算策略：训练容量：仅计算"已完成"组。左右模式：支持"记录左右"勾选，公式：$Σ ((\text{左重量} + \text{右重量}) \times \text{次数})$。
自动化管理：每日 0:00 自动清空"今日训练"列表，数据静默同步至"月视图"。

2. 交互需求
2.1 训练模块 (Training)
多级导航：顶部 Tab 切换【今日训练】、【月视图回顾】、【动作库】。
视觉增强：动作卡片左侧 Icon 统一采用配置好的 GIF 动图。

【今日训练更新】
- 响应式布局：适配小米15等不同手机尺寸，支持 safe-area-inset
- 默认页面：打开 App 默认显示今日健身 Tab
- 当无今日训练时，显示【开始训练】按钮，点击直接跳转到动作库
- 删除动作无需确认弹窗，直接删除
- 动作库支持点击勾选添加到今日训练（绿色对勾标记已选）
- 今日训练标题自动按所选第一个动作的肌肉群命名（如第一个是引体向上→练背日）
- 每组输入框带单位：kg、次，确保数值可见
- 输入框点击打开键盘，inputMode 设为 decimal/numeric
- 左右模式优化：调整左右重量输入框宽度，确保在小米15等窄屏设备上正常显示
- 内置动作库：深蹲、引体向上、卧推
- 动作历史：点击"历史"图标查看动作历史记录（圆角方框逆序流），预置深蹲历史数据用于测试

【月视图回顾更新】
- 日历周首为周日
- 单元格内容：日期、总容量、训练部位、有氧名称
- 支持左右滑动切换月份
- 详情弹窗：点击日期打开详情弹窗
  - 支持点击"编辑"使静态数值转为输入框
  - 点击"完成"保存
  - 删除需二次确认
- 预置历史数据（2026年3月）用于测试

2.2 身体模块 (Body)
动态控制轴：顶部指标（体重、腰围、臂围等）采用横排可拖动单元格。
目标管理：指标下方外显 "目标值"输入框，实时修改并作为基准线反映在折线图中。
图表交互：折线图支持双指缩放，X 轴最小需支持查看 1年内 历史点位数值。
影像中心：照片按上传时间逆序排列。支持单次最多上传 9张。
拼图对比：支持勾选 2-4 张历史照片生成带日期标注的对比图，并保存至相册。

2.3 知识模块 (Knowledge)
文案视觉：左上角固定品牌标语 "自律给我自由"。
结构化目录：文件夹支持点击下拉展开/收起，展开态不随外部点击自动关闭。
智能编辑：支持 Markdown 实时转化（图片、表格、[[双链]] 引用），右下角 "+" 号支持在当前选中文件夹下智能新增。

3. 技术规范
UI 开发：使用 Tailwind CSS，遵循 ui_template.tsx 视觉风格。
视觉标准：品牌色：Vibe Green (#10B981)。组件高度：交互组件（输入框、按钮等）统一为 40px。容器方案：全量使用 圆角方框 (Border Radius: 12px)。
响应式适配：容器不硬编码尺寸，支持 max-width/max-height 适配不同手机长宽，开发环境显示边框便于调试。
数据迁移：支持全量数据一键导出/导入迁移包。
配置同步：动作库、监控指标、知识条目支持后台 JSON/CMS 配置化更新。

4. 详细功能模块拆解
4.1 训练 (Training)
今日训练：支持长按拖动动作排序。点击更多可查看【动作历史】（圆角方框逆序流）。
动作库：移除搜索框右侧 "+" 号。点击动作出现绿色对勾即加入今日计划，再次点击取消。

4.2 身体 (Body)
导航简化：移除左上角返回按钮。
历史管理：列表移除"手动录入"字样。支持编辑（回填弹窗）与删除（二次确认）。

4.3 知识 (Knowledge)
编辑器：类似系统备忘录，支持标题+正文。

5. 页面结构清单
Tab 1: 训练 (今日录入 / 月视图 / 动作库选择 / 动作历史详情)
Tab 2: 身体 (访问密码页 / 指标趋势图 / 历史记录列表 / 照片流与拼图对比)
Tab 3: 知识 (文件夹层级列表 / Markdown 笔记编辑器)
